## Projetos em destaque

### AtlasBank

Sistema bancário full-stack com front-end, back-end, banco em nuvem, documentação Swagger, deploy e versão mobile via WebView.

Principais funcionalidades:

- cadastro, login e autenticação JWT;
- verificação de e-mail com apoio de RPA;
- conta digital, saldo e extrato;
- Pix, boletos e cobranças;
- cartão de crédito, compras e faturas;
- empréstimos e investimentos;
- API documentada com Swagger;
- deploy com Render e Vercel.

Repositórios:

- [Back-end AtlasBank](https://github.com/NinicolasGTH/BackEnd-Banco)
- [Front-end AtlasBank](https://github.com/NinicolasGTH/FrontEnd-Banco)
- [Mobile AtlasBank](https://github.com/NinicolasGTH/Atlas-App-Mobile)
